FEATURES
Automatic Update Checking
On every startup BeamSkin Studio checks GitHub for a newer version and notifies you if one is available.
Mod Support
Basic mod support is included, allowing custom vehicle skins from supported mods — coverage is still expanding.
Colorable Skins
When enabled, the user selects a base color PNG and a color palette map — BeamSkin Studio takes care of the rest.
Material File Editing
Lets the user adjust material values such as metallic, clearcoat, and other surface properties through a simple interface.
Saved Configurations
Users can bundle a saved in-game vehicle configuration — all the tuning and setup they've done to a car — into their mod, optionally with a preview image.
Multiple Languages
The interface supports multiple languages — switch at any time from the settings panel without restarting.
Save / Load Projects
Save your current skin setup as a project file and reload it later to pick up exactly where you left off.

Reflective Map Support
Users will be able to add a reflectivity map to their skin, making specific areas reflective — useful for things like police car decals or chrome accents.

Body Variant Support
Skinning support for cargo, box truck, and ambulance body variants.
V.0.8.0.Beta
Migrated to PySide6
The UI framework is being moved from CustomTkinter to PySide6 — enabling a more modern, flexible interface and better long-term maintainability.
V.0.8.0.Beta
Unpacked Mod Toggle
A toggle to export the mod as a plain folder instead of a ZIP file — useful for testing or editing the mod directly without extracting it first.
V.0.8.0.Beta
Auto Mod Import
Point BeamSkin Studio at a mod and it will automatically scan and extract everything needed — vehicle JSON, JBeam files, and the vehicle's default preview image used in the car list and hover preview — then add the modded vehicle ready to skin. No more manually hunting through mod folders for the right files.
V.0.8.0.Beta
Improved Save / Load
Saving a project now remembers its location automatically. Loading opens a built-in menu listing all your saved projects in one place — no more navigating through File Explorer to track down a save file. Pick a save from the list and you're straight back in.
INSTALLATION GUIDE
01
Download
Click the Windows Download button above to grab the latest release ZIP from GitHub.
02
Extract ZIP
Right-click the downloaded ZIP and choose Extract All. Pick any folder you want BeamSkin Studio to live in.
💡 Avoid paths with spaces or special characters (e.g. C:\Tools\BeamSkin works great).
03
Run Install.bat
Open the extracted folder and double-click Install.bat. It should automatically install all required dependencies.
⚠️ If Windows SmartScreen appears, click More info → Run anyway. This is a false positive common with Python apps.
04
First Launch
Once installation finishes, BeamSkin Studio will launch automatically for the first time.
05
Future Launches
Use BeamSkin Studio.bat to start the app from now on — it skips the installer for a faster startup.
💡 You can create a shortcut to this .bat file on your desktop for quick access.
Linux — heads up: The Linux version is untested and may have issues. If you run into a problem, please report it on the Linux GitHub Issues page. The installer script supports Ubuntu / Debian, Fedora / RHEL, Arch / Manjaro, and openSUSE.
01
Download
Click the Linux Download button above to grab the latest release ZIP from GitHub.
02
Extract ZIP
Right-click and extract with your file manager, or run in a terminal:

unzip BeamSkin-Studio-Linux-*.zip
💡 Avoid paths with spaces. A location like ~/Tools/BeamSkin works well.
03
Run the Installer
Open a terminal inside the extracted folder and run:

chmod +x install_linux.sh
./install_linux.sh

The script automatically installs Python 3, pip, tkinter, and all required Python packages for your distro.
⚠️ You may be prompted for your sudo password to install system packages — this is expected.
04
First Launch
Once the installer finishes it will ask if you want to launch BeamSkin Studio immediately. Type y and press Enter, or launch it yourself anytime with:

./beamskin_studio.sh
OPTIONAL — DESKTOP SHORTCUT
Optional
Edit the .desktop File
Open beamskin-studio.desktop in a text editor. Replace the two /path/to/BeamSkin-Studio placeholders with the actual path to your extracted folder.
💡 Run pwd inside the folder to see the full path.
Optional
Install the Shortcut
Copy the file to your local applications folder so it appears in your app launcher:

cp beamskin-studio.desktop ~/.local/share/applications/
Optional
Future Launches
Search for BeamSkin Studio in your application launcher, or always use the terminal launcher:

./beamskin_studio.sh
💡 You can also create a symbolic link on your desktop pointing to beamskin_studio.sh.
01
Install Python
Download and install Python 3.9 or newer (3.12 recommended) from python.org.
⚠️ On the installer screen, check "Add Python to PATH" before clicking Install Now — this is required.
02
Open a Terminal
Open Command Prompt or PowerShell. On Linux, open your terminal emulator. Navigate to the BeamSkin Studio folder with cd.
💡 In Windows Explorer, type cmd in the address bar while inside the folder to open a prompt there.
03
Install Dependencies
Run the following command to install all required packages at once:

pip install -r requirements.txt
💡 If pip isn't recognised, try python -m pip install -r requirements.txt instead.
WHAT GETS INSTALLED
PySide6 ≥ 6.5.0
The Qt-based GUI framework powering the entire interface. Replaces CustomTkinter as of v0.8.0.
Pillow ≥ 10.0.0
Image processing library used for skin texture manipulation and colorable skin generation.
requests ≥ 2.31.0
HTTP library used for the update checker and online tab features.
pywin32 ≥ 306
Windows-only. Improves taskbar icon grouping and file dialog integration. Skipped automatically on Linux.
Linux note — PySide6 bundles its own Qt libraries so no system Qt packages are needed. You may still need libGL for rendering: sudo apt install libgl1 (Debian/Ubuntu)  ·  sudo dnf install mesa-libGL (Fedora)  ·  sudo pacman -S mesa (Arch)
Antivirus / Windows Defender warnings
Virus scanners including Windows Defender, Windows SmartScreen, and other antivirus software may flag the app — this is a false positive common with Python applications and does not indicate malware.

• On SmartScreen: click More info → Run anyway.
• In your antivirus: add the BeamSkin Studio folder as an exception / exclusion.
ModuleNotFoundError on launch
A missing module means the installation didn't complete correctly.

• Windows: Re-run Install.bat and let it finish without interruption. Make sure you're launching via BeamSkin Studio.bat, not main.py directly.
• Linux: Re-run ./install_linux.sh and check for any errors. You can also install the missing package manually with pip install --user <package-name>.
Linux: tkinter not found / GUI fails to open
tkinter is not always bundled with Python on Linux and must be installed separately via your package manager:

• Ubuntu / Debian / Mint: sudo apt install python3-tk
• Fedora / RHEL: sudo dnf install python3-tkinter
• Arch / Manjaro: sudo pacman -S tk
• openSUSE: sudo zypper install python3-tk

The installer script handles this automatically, but if you skipped it, install tkinter manually using the command above.
Linux: permission denied when running scripts
The shell scripts need execute permission before they can be run. Fix this with:

chmod +x install_linux.sh beamskin_studio.sh

Then run them with ./install_linux.sh or ./beamskin_studio.sh.
Everything keeps failing — fresh start
If repeated fixes don't help, do a clean reinstall:

1. Delete the downloaded ZIP and the extracted folder entirely.
2. Re-download a fresh copy from the button above.
3. Extract to a new location and run the installer again (Install.bat on Windows, ./install_linux.sh on Linux).
Other errors or bugs
If you hit an issue that isn't covered here, please submit a report on GitHub so it can be investigated:

• Windows Issues  ·  Linux Issues

Include any error messages or console output — it makes fixing things much faster.
